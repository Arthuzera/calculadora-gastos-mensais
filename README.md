# Gerenciador de Tarefas

Este é um programa simples em Python para gerenciar tarefas pessoais. Ele permite adicionar, visualizar, marcar como concluídas e remover tarefas.

## Funcionalidades
- Adicionar tarefas
- Visualizar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Como usar
Execute o programa no terminal e siga as instruções do menu.

## Requisitos
- Python 3.x
